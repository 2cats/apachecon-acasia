API 正在成为现代企业互连系统、应用程序和服务的支柱，为客户提供全新的创新体验并加快上市速度。 随着 API 对于企业变得越来越重要，对有效 API 管理和安全的需求也变得越来越强烈。该主题会将重点关注 API 安全性和可扩展性的关键主题，为与会者提供部署安全且可扩展的 API 管理解决方案所需的最佳实践和策略。
